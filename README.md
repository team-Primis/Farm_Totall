<h3>🌱게임에 대한 정보</h3>
이 게임은, 막 유니티 기초 공부를 끝내고 실제 게임을 제작하고 싶어서 만들게 된 첫 게임 프로젝트입니다.<br>
따라서 기본적인 기능 ( 식물 재배 및 수확, 닭 구매 및 알 수확, 인벤토리 자판기등 구현) 구현에 초점을 두었습니다.<br>
게임의 재미 요소 등은 추후 업데이트를 통해 추가가 될 수도 있으나 시기는 미정입니다!<br>
